Significance tests

This section will describe the comparisons between sections and their practical and statistical significance.
For all tests, independence and normality is assumed.

ANOVA Results (global significance test)

ANOVA is a difference of means test for multiple groups. It gives a global p-value, which if significant, means that it's likely that a students average score is effected by the students section, in general.
Note that ANOVA assumes normality, and equal variance in each group. Especially the second criteria could be false for student scores, and if so, ANOVA results should not be trusted blindly.

P-value: <0.00005

(With significance level = 0.05)
We reject the null hypothesis. The different groups likely don't share the same true mean. There's a <0.005% chance of a Type I error.

F-statistic: 13.119

A larger F-statistic means more difference between groups.
An F-stat of 13.119 means that the variance between sample means is 13.119 times the variance within samples.

Summary statistics (Pairwise significance tests)

This table summarizes the information from pairwise t-tests between each section. The detailed information of each t-test can be found below this table.
