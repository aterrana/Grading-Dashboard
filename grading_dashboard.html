Section Comparisons

This section describes the comparisons between sections and their practical and statistical significance.
For all tests, independence and normality is assumed.

ANOVA Results (global significance test)

ANOVA is a difference of means test for multiple groups. It gives a global p-value, which if significant, means that it's likely that there is a general association between student average score and section.
Note that ANOVA assumes normality, and equal variance in each group. Especially the second criteria could be false for student scores, and if so, ANOVA results should not be trusted blindly.

P-value: 0.0017

(With significance level = 0.05)
We reject the null hypothesis. The different groups likely don't share the same true mean. There's a 0.17% chance of a Type I error.

F-statistic: 6.482

A larger F-statistic means more difference between groups.
An F-stat of 6.482 means that the variance between sample means is 6.482 times the variance within samples.
